"# AI Ассистент DDC KZ - Настройка

## 🎯 Архитектура

### Общая логика AI

**`ai-context-helper.ts`** - общий модуль для формирования AI контекста:
- ✅ Определение языка (`getLangCode`)
- ✅ Формирование контекста (`createContext`)
- ✅ Генерация system prompt (`generateSystemPrompt`)
- ✅ Стандартные параметры Groq API (`DEFAULT_GROQ_PARAMS`)
- ✅ Ответы по умолчанию (`getDefaultReply`)

### Локальная среда (Angular App)
- **LocalAiService**: Использует `ai-context-helper.ts` для формирования промпта
- **ApiKeySettingsComponent**: Интерфейс для ввода API ключа (сворачивается/разворачивается)
- **Хранение**: API ключ сохраняется в localStorage браузера
- **Отображение**: Только в локальной среде (localhost)

### Продуктивная среда (Netlify Function)
- **chat.ts**: Использует `ai-context-helper.ts` для формирования промпта
- **Хранение**: API ключ в переменных окружения Netlify
- **Отображение**: Настройки API ключа НЕ отображаются

## 📋 Настройка локальной разработки

### 1. Получите Groq API ключ

1. Зайдите на [console.groq.com](https://console.groq.com/)
2. Войдите или зарегистрируйтесь
3. Перейдите в раздел **API Keys**
4. Создайте новый ключ
5. Скопируйте ключ (начинается с `gsk_`)

### 2. Введите ключ в приложении

1. Запустите приложение: `npm start`
2. Откройте браузер: `http://localhost:4200`
3. Прокрутите вниз до секции **"🔑 Groq API Ключ"**
4. Нажмите на заголовок чтобы развернуть
5. Введите ваш ключ и нажмите **"Сохранить"**
6. Ключ сохранится в localStorage вашего браузера
7. Нажмите на заголовок чтобы свернуть

### 3. Проверка работы

1. Откройте AI ассистент (иконка в правом нижнем углу)
2. Задайте вопрос: "Кто директор DDC KZ?"
3. Должен прийти ответ от AI модели

## 🔒 Безопасность

- ✅ `.env` файл не коммитится в Git (в `.gitignore`)
- ✅ API ключ хранится только в браузере (localStorage)
- ✅ На Netlify ключ хранится в переменных окружения
- ✅ Настройки API ключа отображаются ТОЛЬКО в локальной среде
- ✅ Компонент можно свернуть/развернуть
- ❌ НЕ храните ключ в коде
