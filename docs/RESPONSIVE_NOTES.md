# Responsive update

Изменены только SCSS. HTML менять не нужно.

## Что сделано

- `home(3).scss`: главный `home hero` адаптирован под 1920×1080 и ноутбуки с небольшой высотой.
- На ширине до `1680px`/высоте до `900px` уменьшаются отступы, orbit, карточки, command panel и нижние карточки.
- До `1366px` hero становится 2-колоночным, правый command panel уходит вниз одной строкой.
- До `1100px` hero становится одной колонкой.
- До `640px` orbit-схема превращается в обычный список карточек, чтобы не ломалась на телефонах.
- `styles.scss`: добавлен общий responsive layer для всех страниц: контейнеры, заголовки, секции, карточки, фильтры, модалки, кнопки.

## Замены классов в HTML

Не требуются.

## Проверить после вставки

1. Главная на 1920×1080: hero должен помещаться без ощущения, что всё слишком крупное.
2. 1366×768: command panel должен уходить под hero, а не сжимать orbit.
3. 768px и 390px: orbit должен стать списком карточек.
4. Модалки проектов/вакансий на мобилке должны скроллиться всей страницей, крестик не должен перекрываться.
